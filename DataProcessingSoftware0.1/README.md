##QT写的串口调试上位机（带波形显示）
****
###软件主要功能
1. 可以实时接收显示数据，同时也可以向单片机发送数据；
2. 可以读取文件中数据，不断发送给下位机；
3. 可以实时的显示波形；并通过一些软件算法对采集的信号进行滤波；
4. 可以自动定时发送数据给下位机。
